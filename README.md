# NOCS (New/Old Clothes Sharing)

## 팀 프로젝트 참가자
- 정민서: Front-End 담당자 (웹앱 페이지 생성 및 디자인), 서버와 통신 연결
- 여강인: Back-End 담당자 (GCP로 서버 생성 및 관리, API 생성 및 관리)
- 허진수: 프로젝트 진행 담당자 (Github와 Notion 관리, Front-End 일부 도움)
